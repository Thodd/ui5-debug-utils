Available APIs under the global _ui5
If not present, call _ui5dbg.init().
require(string|string[]):Promise<any[]>
Reqires the given module or list of modules, resolves with the resp. module's content e.g. const [a, b] = await _ui5.require(["sap/ui/core/Theming", "sap/ui/core/Core"])
control(HTMLElement):Element
retrieves the Element associated with the given HTMLElement
byId(sap.ui.core.ID):Element
retrieves the Element with the given ID
owner(Element|HTMLElement):Component
retrieves the owner Component of the given Element
probe(string):any
Probes for a module by the given name. Shorthand for sap.ui.require(string)
